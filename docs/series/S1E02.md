# S1E02 — Data Model: JSONB + Zod

**Why this matters:** Flexible schema supports iteration without heavy migrations.

## What we built
- `ResumeDoc` Zod schema
- `resumes` JSONB table & seed row (`slug=main`)

## Key decisions
- JSONB over relational schema (ADR-004)
- Shared types package (ADR-013)
