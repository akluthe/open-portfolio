# S1E04 — Web SSR from API + Print CSS

**Why this matters:** Fast, SEO-friendly rendering and a clean print layout for recruiters.

## Built
- `/r/main` server component fetch
- Print stylesheet (`@media print`)

## Decisions
- SSR via Next.js App Router (ADR-003)
- Print-first PDF strategy (ADR-015)

## Run
```bash
RESUME_API=http://localhost:8080 npm run dev --prefix apps/web
```

## Demo
1) Change seed JSON; refresh `/r/main`.
2) Print preview shows a tight 1–2 page resume.
