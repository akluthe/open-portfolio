# S1E05 — Local Admin Editing (Dev-Only)

**Why this matters:** Let the platform live and evolve without manual DB edits.

## Built
- `/admin/main` form for basics/experience
- `PUT /resumes/:slug` upsert
- Dev-only feature flag

## Decisions
- Dev-only first to reduce auth friction (ADR-011)

## Run
```bash
make up
npm run dev --prefix apps/web
dotnet run --project apps/api-resume
```

## Demo
1) Edit headline; save.
2) Refresh public `/r/main` to see change.
3) Toggle dev-only flag off/on.
